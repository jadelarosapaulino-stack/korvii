# Prompt - Identidad de Marca Korvi

Usa este prompt cuando necesites generar o extender piezas de marca para Korvi.

```text
Crea una identidad de marca completa para un proyecto llamado KORVI.

Contexto:
KORVI es una plataforma tecnologica enfocada en seguridad vial, movilidad inteligente, analitica de riesgos e InsurTech. El producto ayuda a ciudadanos, aseguradoras y gobiernos a reducir accidentes, reportar incidentes viales, analizar zonas de riesgo y mejorar la gestion de reclamaciones mediante datos en tiempo real.

Objetivo:
Generar una identidad de marca profesional, moderna y escalable para una startup tecnologica.

Incluye:
1. Nombre de marca: KORVI
2. Significado conceptual del nombre
3. Slogan principal
4. Variaciones de tagline
5. Mision
6. Vision
7. Valores de marca
8. Personalidad de marca
9. Publico objetivo
10. Propuesta de valor
11. Mensajes clave por segmento:
   - Ciudadanos
   - Aseguradoras
   - Gobierno
12. Paleta de colores con codigos HEX
13. Tipografias recomendadas
14. Concepto de logo
15. Ecosistema de productos:
   - KORVI Drive
   - KORVI Insight
   - KORVI Risk
   - KORVI AI
16. Elevator pitch
17. Tono de comunicacion
18. Ejemplos de uso de marca en app, dashboard y presentacion.

Estilo:
Moderno, tecnologico, limpio, confiable, tipo startup SaaS / InsurTech / Smart Mobility.

Formato de salida:
Documento estructurado con titulos, subtitulos y bullets claros.
```
