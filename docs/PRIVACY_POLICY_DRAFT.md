Esta politica describe como Ruta Segura / KORVI Drive trata la informacion de las personas que usan la aplicacion web y movil.

## Datos Que Recopilamos

Podemos recopilar:

- Datos de cuenta: nombre, correo electronico, provincia, municipio y tipo de usuario vial.
- Datos de autenticacion: tokens de sesion y, si se usa login social, identificadores provistos por Google o Facebook.
- Reportes ciudadanos: titulo, descripcion, categoria, nivel de riesgo, ubicacion, direccion aproximada, fotos adjuntas y fecha de creacion.
- Ubicacion: coordenadas usadas para crear reportes, mostrar riesgos cercanos, registrar llamadas de emergencia o activar funciones de mapa.
- Actividad dentro de la app: vistas de pantalla, acciones de navegacion y eventos necesarios para mejorar el servicio.
- Progreso educativo: lecciones vistas, puntajes y estado de avance.
- Datos tecnicos: direccion IP, navegador, dispositivo, errores y logs operativos.

## Para Que Usamos Los Datos

Usamos la informacion para:

- Crear y mostrar reportes de seguridad vial.
- Alertar sobre riesgos cercanos.
- Gestionar cuentas, autenticacion y recuperacion de acceso.
- Mostrar contenido educativo y medir progreso.
- Mejorar la seguridad, estabilidad y calidad del servicio.
- Facilitar revision operativa por moderadores o instituciones autorizadas.

## Fotos Y Ubicacion

Las fotos y ubicaciones enviadas por usuarios pueden ser visibles para operadores autorizados y, segun la configuracion del sistema, para otros usuarios dentro del mapa o listado de reportes. No publiques imagenes que contengan datos personales innecesarios, menores de edad identificables o informacion sensible.

La funcion de emergencia registra la intencion de llamar al 911 y puede incluir ubicacion si esta disponible. La aplicacion no sustituye el servicio oficial de emergencias.

## Con Quien Compartimos Datos

Podemos compartir informacion limitada con:

- Instituciones o moderadores autorizados para gestionar reportes.
- Proveedores de infraestructura, base de datos, almacenamiento, correo, mapas, analitica, inteligencia artificial o notificaciones.
- Autoridades competentes cuando sea requerido por ley o para proteger la seguridad publica.

No vendemos datos personales.

## Conservacion

Conservamos los datos mientras sean necesarios para operar el servicio, mantener historial de reportes, cumplir obligaciones legales o resolver incidentes de seguridad. Puedes solicitar revision, actualizacion o eliminacion de datos cuando aplique.

## Seguridad

Aplicamos controles razonables como HTTPS, validacion de entradas, roles de acceso, almacenamiento protegido y monitoreo. Ningun sistema es completamente infalible.

## Derechos Del Usuario

Puedes solicitar:

- Acceso a tus datos.
- Correccion de informacion incorrecta.
- Eliminacion de tu cuenta o datos personales, cuando sea legal y tecnicamente posible.
- Informacion sobre el tratamiento de tus datos.
