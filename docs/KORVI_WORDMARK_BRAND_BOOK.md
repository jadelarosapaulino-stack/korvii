# KORVI Wordmark Brand Book

## Brand Core

KORVI is a mobility intelligence platform for road safety, risk analytics, InsurTech, citizen reporting, AI-assisted decisioning, insurer dashboards, and Smart City operations.

Brand attributes:

- Modern
- Premium
- Technical
- Minimal

Brand promise:

```text
KORVI
Smart Mobility Platform
```

Recommended positioning line:

```text

```

## Visual System

The logo system is wordmark-only. Do not use shields, road graphics, icons, symbols, pins, crests, isotopes, badges, or illustrative marks as part of the logo.

Primary palette:

| Token | Hex | Usage |
| --- | --- | --- |
| Primary | `#0B1F3A` | Main wordmark, enterprise surfaces, dashboard headers |
| Secondary | `#00C2A8` | Intelligence accents, active states, data highlights |
| Accent | `#FF6B35` | CTAs, warnings, conversion moments, loader pulse |
| Background | `#F8FAFC` | App backgrounds, landing page base, PDF page background |
| White | `#FFFFFF` | Dark-mode wordmarks and high-contrast surfaces |
| Muted Text | `#64748B` | Taglines, metadata, secondary labels |
| Border | `#E2E8F0` | Separation lines and quiet UI structure |

Typography:

- Primary logo type: Sora
- Secondary UI/brand type: Inter
- Alternative commercial/marketing type: Poppins

Recommended logo weight:

- `KORVI`: Sora ExtraBold / 800
- Product suffixes: Sora SemiBold / 600
- Taglines: Inter Medium / 500
- Minimal forms: Sora Black / 900

Letter spacing:

- Main wordmark: `0.02em`
- Corporate subtitle: `0.08em`
- Minimal forms: `0.04em`

## UX/UI Guidance

Navbar:

- Use `KORVI` or `KORVI Insight`.
- Keep height between 32 and 40 px.
- Do not add icons next to the logo.
- Keep product suffix optional; avoid long lockups in tight nav.

Sidebar:

- Expanded: `KORVI Admin`, `KORVI Insight`, or `KORVI Risk`.
- Collapsed: `KRV`.
- Minimum clear space: 16 px left and right.

Mobile:

- Login/splash: `KORVI Drive`.
- App bar: `KORVI` only.
- Use centered wordmark on auth screens and left-aligned wordmark inside app shell.

Splash screen:

- Preferred: dark background `#0B1F3A`, white `KORVI`, teal tagline.
- Avoid animated icons. Use text fade/pulse only.

## Accessibility

Contrast recommendations:

- `#0B1F3A` on `#F8FAFC`: passes WCAG AA/AAA for normal text.
- `#FFFFFF` on `#0B1F3A`: passes WCAG AA/AAA.
- `#00C2A8` on `#0B1F3A`: acceptable for large text and accents; avoid as small body copy.
- `#FF6B35` on `#F8FAFC`: use for CTAs with sufficient size/weight; avoid tiny text.

Rules:

- Do not place teal logo text over white when below 14 px.
- Do not place orange logo text over white for small subtitles.
- On dark backgrounds, keep the main wordmark white.
- For legal and accessibility-safe contexts, use monochrome navy or white.

## Final Governance

Do:

- Use only typographic wordmarks.
- Keep suffixes short.
- Use navy for trust, teal for intelligence, orange for action.
- Use Sora for brand, Inter for UI.
- Use one product suffix per context.

Do not:

- Add road, shield, pin, car, AI, government, or insurance symbols to the logo.
- Stretch or condense the wordmark.
- Use gradients inside wordmark text.
- Use more than two colors inside a lockup.
- Create independent sub-brand icons.

KORVI should behave like a premium SaaS brand: restrained, repeatable, legible, and modular.
